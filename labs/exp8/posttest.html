Post Test

  1. Software or Hardware used to prevent unauthorized access to a network?
    2. a. Hub
    b. Firewall
    c. Switch
    d. Network Driver

  2. What is an important piece of computer hardware that is necessary for a computer to connect to a network?
    3. a. Proxy
    b. Network Interface Card
    c. Switch
    d. Bridge

  3. Which devices sole purpose is to strengthen and retransmit the signal it receives?
    4. a. Repeater
    b. Bridge
    c. Modem
    d. Gateway

  4. Packets in the data link layer are also called Frames.
    5. a. True
    b. False

  5. Which one of the following network devices has only two ports?
    6. a. Router
    b. Bridge
    c. Both a and b
    d. None of the above

  6. Which layer is responsible for converting data packets from the Data Link layer into electrical signals?
    7. a. Data Link Layer
    b. Network Layer
    c. Transport Layer
    d. Physical Layer

  7. How many types of routers are there?
    8. a. 1
    b. 2
    c. 3
    d. 4

  8. Which one of the following divides the larger networks into smaller networks or smaller sections?
    9. a. Repeater
    b. Bridge
    c. Hub
    d. Gateway

  9. Which one of the following network devices is the broadcast device?
    10. a. Hub
    b. Repeater
    c. Both a and b
    d. None of the above

  10. Which one of the following bridge services is connectionless?
    11. a. Transparent bridge
    b. Source routing bridge
    c. Both a and b
    d. None of the above
