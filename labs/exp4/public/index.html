Follow these steps:
1. Click on New Graph button.
2. After new graph is added, complete the table manually (entries in upper case) and click the Check button.
3. For adding an extra edge specify the source and destination along with the weight to connect the two nodes and click the Submit button.
4. As new edges are added, update the table accordingly and click the Check button again.
5. For sending the data packet, provide the source node and destination node in the space provided and click the Send Message button.
6. The best path will be chosen and displayed using red lines.
