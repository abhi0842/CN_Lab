Implementation of Line Encoding Schemes

Post Test

  1. Which of the following characteristics are true for Differential manchester.
    C1: Three levels.
    C2: No DC component.
    C3: Good synchronization.
    2. a. C1 and C2
    b. C2 and C3
    c. C1 and C3
    d. None of the above

  2. Which of the following is the correct input string for the following waveform generated using Unipolar-RZ.

    3. a. 11101
    b. 10101
    c. 11111
    d. 00110

  3. In which line encoding scheme high volt is assigned for logical 0 bit and low volt is assigned for logical 1.
    4. a. NRZ-I
    b. NRZ-L
    c. AMI
    d. Manchester

  4. Which of the following is the correct input string for the following waveform (in NRZ-L).

    5. a. 10110001
    b. 10010110
    c. 01010110
    d. 01001110

  5. The minimum bandwidth of Manchester and differential Manchester is ______ that of NRZ.
    6. a. the same as
    b. twice
    c. thrice
    d. none of the above

  6. Which of the following is the correct input string for the following waveform generated using NRZ-I. Assume low voltage at the beginning.

    7. a. 11101
    b. 10101
    c. 11111
    d. 00110

  7. Which of the following techniques is best at self-synchronization.
    8. a. Manchester
    b. NRZ-L
    c. Unipolar RZ
    d. All of the above

  8. Line coding converts a sequence of ___________.
    9. a. frames to digital signal
    b. bits to a digital signal
    c. bytes to digital signal
    d. packets of data to digital signals

  9. Which of the following techniques have no DC component?
    10. a. Polar NRZ-I
    b. Pseudoternary
    c. Manchester
    d. Both b and c

  10. Which of the following is the correct input string for the following waveform generated using Pseudoternary. Assume the most recent preceding 1 bit has negative voltage.

    11. a. 10101010101
    b. 11001110010
    c. 01001100011
    d. None of the above
