Instructions

    1- Send Request :

  • Click the "Send Request" button to open a text box where you can enter your name, email, and message. After entering the required information, press Enter to send it.

  • This action triggers the animation of packets and text from the Presentation Tier to the server tier.

  • The entered details will be displayed in a table at the right corner of the page.

    • 2- Send Response :

  • After sending the request, Click the "Send Response"button to simulate the server processing the request and sending a response back to the client. This action is only available after sending a request.

  • The packets will move from the server tier back to the client tier, indicating the response flow.

    • 3- Fetch Data:

  • To fetch details associated with your email ID, enter your email address in the designated field and click the "Fetch Data" button.

  • This action will retrieve and display your previously submitted information in a table format at the right corner of the screen.

    • 3- Click the "Reset" button to refresh the page and reset all animations and data.

    4- Click the "Print" button to print the current state of the page.
