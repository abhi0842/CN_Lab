Circuit Switching: Setup, Transmission, and Teardown

Pre Test

1. What is circuit switching?

a) a method of transmitting data in packets
b) a dedicated communication path between two stations
c) a technique that stores data at intermediate nodes
d) a wireless communication method

2.) _______ switching assumes that the data rate in both directions is the same.

a) Circuit
b) Packet
c) Message
d) both b and c

3. When are resources [like lines] used in circuit switching?

a) only sometimes
b) all the time
c) randomly
d) only when connection is made

4. The best use of circuit switching is for?

a) e-mail
b) chat
c) voice calls
d) file transfer

5. Why is Circuit Switching less suitable for modern internet usage?

a) It is compatible with all cloud services
b) It uses packet prioritization
c) It supports high scalability
d) It requires pre-established circuits and wastes bandwidth
